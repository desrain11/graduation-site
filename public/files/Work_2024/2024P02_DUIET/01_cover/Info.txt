Title: Introducing HERMITS to the Kitchen: An Interactive Toolkit for Beginners

Time：2024.11 - 2025.01

Project Type: Design Research Project

Role: Interaction Design, Prototype Development, User Testing

Location: TU Campus, Eindhoven, Netherlands

Client: -

EA Involved：Technology & Realization/Creativity & Aesthetics/User & Society/Math, Data & Computing

Abstract: This project explores how a tangible interactive toolkit can support beginner cooks through step-by-step cooking guidance. Inspired by the HERMITS [1] modular tangible interface, the toolkit consists of three physical cases for recipe guidance, ingredient weighing, and timing. Rather than automating the cooking process, the project focuses on interactive learning by encouraging users to physically engage with cooking tasks and receive real-time feedback. Through 3D-printed prototypes, LED-based Wizard of Oz simulation, and user testing with six participants, this project investigated how tangible user interfaces can improve clarity, engagement, and confidence for adolescents and young adults learning to cook. The findings highlight the potential of modular interaction for hands-on learning, while also revealing the need for clearer feedback, personalized guidance, and visual or auditory instruction.

[1] Nakagaki, K., Leong, J., Tappa, J.L., Wilbert, J. and Ishii, H. 2020. HERMITS: Dynamically Reconfiguring the Interactivity of Self-propelled TUIs with Mechanical Shell Add-ons. Proceedings of the 33rd Annual ACM Symposium on User Interface Software and Technology (Virtual Event USA, Oct. 2020), 882–896.
