## General

This UX challenge focused on improving the first-time user experience of Bathbobber, an existing cold-water therapy product. The project analyzed how new users understand, set up, and interact with the system, and redesigned the onboarding flow to reduce confusion and create a smoother connection between the app, device, and first-use experience.

## Contribution to PI&V

- The UX challenge made me more aware that UX when onboarding a new product shapes trust and readiness, not only usability.
- UX became more clearly a bridge between user perspective, technology, and stakeholder expectations.

## Involved Expertise Areas

- **U&S**: first-time user journey analysis, onboarding pain-point identification, empathy-based UX framing
- **C&A**: onboarding flow redesign, information structuring, app-device touchpoint refinement
